# VideoMeet ‚Äì WebRTC Video Conferencing App

VideoMeet is a modern, full-stack video conferencing web application built with React, Node.js, and WebRTC. It enables seamless, real-time video, audio, and screen-sharing meetings directly in your browser‚Äîno plugins required. Designed for ease of use, privacy, and high-quality group calls.

## üöÄ Features

- **Multi-party video conferencing** with real-time audio/video
- **Screen sharing** support
- **Dynamic meeting rooms** (create/join via unique links)
- **Mute/unmute microphone** and **enable/disable camera** at any time
- **Participant management** (see who is in the call)
- **Responsive UI** for desktop and mobile
- **Persistent user settings** (display name, device preferences)
- **WebRTC TURN/STUN** for NAT traversal and robust connectivity
